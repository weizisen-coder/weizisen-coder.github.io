请问?

如何装这些物品,才能使得背包中的总价值最大,并且不超过背包的最大承载重量?

//每个物品对应的价值及重量
let items = [
  { name: "照明灯", val: 5, weight: 4 },
  { name: "帐篷", val: 8, weight: 10 },
  { name: "饮用水", val: 6, weight: 3 },
];

//最大承载重量
let maxWeight = 10;

//最终背包携带物品

# 什么是算法?

简单来说就是一系列通过计算来解决问题的明确的步骤

生活中处处有算法,比如小学学的九九乘法表,或者你平时在做饭的时候脑子中出现的食谱步骤等等。

在编程中,每一个程序都是一个算法或者每个程序中都包含了许多较小的算法,例如给一个按钮绑定事件,每次点击 number 值就加 1。这其实就是算法。
