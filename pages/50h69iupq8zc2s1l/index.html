# 什么是渐进分析？(推导大 O 符号)

渐进分析是一种描述函数在极限附近的行为的方法,当输入规模足够大与运行时间的增长量相关时,我们就要研究算法的渐进效率

简单来说就是输入规模无限增加,那么算法所消耗的时间是如何随着输入增加而增加的,这个过程就是渐进分析

渐进分析有如下三个步骤:

# 1. 定义函数

  • 计算代码表达式执行的次数
function sumUp(n) {
  //代码表达式1
  let result = 0;
  //代码表达式2
  for (let i = 0; i <= n; i++) {
     //代码表达式3
    result += i;
  }
  //代码表达式4
  return result;
}
//当前函数参数n 并无特定值,所以需要我们定义
设定 n = 1  n = 3 n = 10

结论

通过上述的测试,可以推导出代码表达式执行次数,设次数为 T , 所以 T = 1+ 1 + n + 1 = 3+n = 3+ 1 * n, 虽然这样并不会直接得出当前函数的运行时间,其因为在计算机中影响代码执行时间的因素有很多,假设每行代码花费相同的时间,那么我们就可以通过运行的次数,可以衡量他们的性能。因为运行的次数越多,消耗的时间肯定是越多

注意

至于最终为什么会转换成 3+ 1 * n,是因为假如我们上述函数的代码表达式 3 中在多添加一行代码,如 console.log()。那么最后的公式应该是3+ 2 * n这样我们虽然会得到不一样的线性图形,但是整体走向和趋势是一样的。我们其实更关注的是整体走向和趋势

  • 最后,我们可以推导出一个通用线性函数 T = a * n + b

# 2.找到最快增长项

  • 因为 b 是常数, 所以变化部分 只有 a* n 这一部分,所以 最快增长项为 T = a * n。 n 越大,增长速度越快

# 3.去掉前面的系数

  • 因为我们不关心具体的值,只关心整体趋势和走向. 所以最后推导出 T = n 。及 O(n)代表为线性时间复杂度
